# Industry Research Report Design

## Use Case

用于行业研究、投研报告、咨询汇报、市场进入分析。整体风格偏严肃，强调证据链、结构化判断和可追溯的数据口径。

## Visual System

- 主色：蓝黑 `#111f35`。
- 强调色：铜金 `#b8893a`。
- 风险色：暗红 `#b64a3c`。
- 背景：冷灰白 `#f3f5f7`，表格和卡片使用白色。
- 版式：报告页眉、编号章节、结论条、证据卡、图表区。

## Chart Rules

- 市场规模和渗透率可用双线或柱线组合；若单位不同，需要明确口径。
- 趋势图至少 2 条相关指标，不能为了装饰硬放折线。
- 竞争格局适合矩阵、雷达图或份额卡。
- 结论必须对应前文证据，不写空泛判断。

## Interaction Rules

- 顶部切换：摘要 / 市场 / 结构 / 竞争 / 结论。
- 图表点位提供 tooltip。
- 竞争矩阵支持点击高亮重点象限。
