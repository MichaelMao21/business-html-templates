# Industry Research Report Design

## Use Case

用于行业研究、投研报告、咨询汇报、市场进入分析。新版结构采用 `industry anatomy infographic`：用一个明确的行业主视觉承载研究对象，周围放产业链、关键指标、排行、区域/行业分布和结论路径。

## Visual System

- 主色：黑白报纸风，`#222222` 用于标题、分隔线和章节条。
- 强调色：研究红 `#c9433d`，用于核心数据、callout、进度条和重点路径。
- 背景：暖纸色 `#f8f6f0`，搭配轻微网格，形成研究海报感。
- 主视觉：中央 3D cutaway / anatomy image，适合替换成行业工厂、数据中心、建筑剖面、设备拆解或动物解剖隐喻。
- 结构：左侧产业链步骤，中间行业主视觉，右侧关键驱动和指标，底部从基础设施到解决方案的路径。

## Layout Rules

- 中央主视觉必须一眼能识别行业对象，不能使用抽象几何图替代。
- 左侧产业链建议 4-6 步，描述价值链上下游。
- 右侧只放最关键的 3-4 组指标，避免变成普通 dashboard。
- 底部路径用于总结“从输入到解决方案”的逻辑闭环。
- 如果行业适合动物隐喻，可以使用动物解剖；科技、AI、制造、能源更适合 3D 建筑、工厂或设备剖面。

## Chart Rules

- 指标必须有明确口径说明。
- 排行条形图适合展示 top use cases、popular segments、regional share。
- 中央 callout 不应超过 2 个，否则会遮挡主视觉。
- 产业链步骤不要写成长段解释，每步控制在 1-2 行。

## Interaction Rules

- 页面必须无需构建即可打开。
- 文字可通过共享 editable runtime 编辑、保存和导出。
- 该模板是单页研究海报，不使用顶部 tab；如果需要多页研究报告，应复制本页作为不同章节。
