# Goals Calendar Agenda Design

## Use Case

用于 OKR 制定、月度计划、周会安排、项目节奏管理。页面要让团队快速看懂目标、日期、负责人、优先级和下一步动作。

## Visual System

- 主色：墨绿 `#123b35`，用于标题、导航和关键节点。
- 强调色：珊瑚红 `#e65f4f`，用于高优先级事项。
- 辅助色：蓝色 `#3478c5`、金色 `#d8a23a`、浅薄荷 `#eaf5ef`。
- 结构：月历网格、议程时间轴、目标进度仪表盘。
- 圆角：8px 以下，保持工具型而非营销页。

## Interaction Rules

- 顶部视图切换：总览 / 月历 / 议程 / 复盘。
- 目标完成率用仪表盘，不用装饰性环形图。
- 月历格子可直接替换日期和事项。
- 议程时间轴按时间顺序排列，必须包含主持人、输出物或决策点。
- 动效只用于视图切换、进度条和议程节点进入。
