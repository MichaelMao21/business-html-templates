# Strategy Planning Brief 使用说明

这套模板用于 BLM 风格的战略规划、年度经营规划、战略解码工作坊和管理层汇报。它适合从“差距”出发，依次完成市场洞察、战略意图、战略目标、业务设计和组织解码。

## 推荐使用顺序

1. **总览**：说明公司、业务背景、战略主题和本次汇报结论。
2. **差距**：区分业绩差距和机会差距。
3. **BLM模型**：解释战略侧和执行侧的整体闭环。
4. **洞察**：用市场增长和竞争地位两个维度圈定战略坐标。
5. **意图**：填写使命 Why、愿景 Where、目标 What。
6. **目标**：把三年目标拆成财务、客户、内部运营、学习成长四个维度。
7. **业务设计**：填写客户选择、价值主张、盈利模式、活动范围、持续价值、风险管理。
8. **解码路径**：把战略转成阶段、战役、负责人和复盘节奏。
9. **能力**：用雷达图对比现状能力和目标能力。

## 必须替换的内容

- 公司名称、年份、业务对象。
- 业绩差距和机会差距的数字与解释。
- 市场洞察页的象限判断和四类战略清单。
- 使命、愿景、三年目标示例。
- 四个目标维度的具体目标。
- 业务设计六要素。
- 解码路径中的阶段任务、负责人和关键指标。
- 能力雷达中的能力维度和现状/目标差距。

## 图表使用规则

- 差距图用于解释“现状、目标、愿景”之间的差距，不用于普通业绩复盘。
- 市场洞察矩阵必须同时判断市场增长和竞争地位，不要只凭单一维度下结论。
- 三年目标页适合展示目标分解，不适合放过多过程指标。
- 能力雷达用于多维能力对比，不用于时间序列数据。
- 如果需要趋势数据，优先使用共享 SVG 折线图组件，并确保 2-3 条线具有相同时间轴和可比口径。

## 不适合的场景

- 只需要轻量周会、月会或排期管理。
- 没有明确战略差距，只是做普通业务介绍。
- 需要大量行业研究证据页，但不需要战略解码。
- 偏营销、发布会、品牌故事的展示型页面。
